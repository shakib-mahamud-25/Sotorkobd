# Sotorko (সতর্কো)

A crowdsourced, anonymous safety map for women navigating Dhaka.

## Start here

1. **[SETUP_GUIDE.md](./SETUP_GUIDE.md)** — set up Supabase, Cloudinary, and Vercel from scratch.
2. **[DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)** — get this code onto GitHub and live, plus how to make changes later.
3. **[PROJECT_CONTEXT.md](./PROJECT_CONTEXT.md)** — how the whole system works, why key decisions were made, and a full file map. Read this before asking an AI assistant to build new features, so it has the full picture.

## Local development (optional)

You don't need this to deploy — Vercel builds from GitHub directly. But if you ever want to preview changes locally before committing:

```bash
npm install
cp .env.example .env.local   # then fill in your real values
npm run dev
```

Visit `http://localhost:3000`.
